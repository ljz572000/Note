# 文件结构
```
├── build.gradle //配置当前项目的构建描述文件
├── gradle
│   └── wrapper
│       ├── gradle-wrapper.jar  //Gradle Wrapper 可执行JAR
│       └── gradle-wrapper.properties  //Gradle Wrapper 配置属性
├── gradlew  //对于Unix系系统Gradle Wrapper解释
├── gradlew.bat  //对于Windos的Gradle Wrapper解释
└── settings.gradle //配置Gradle build的Gradle设置文件
```
